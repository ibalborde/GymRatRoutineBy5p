# GymRat Routine by 5p

A simple iOS app to track progress on a 5-day gym split routine. Built with SwiftUI and SwiftData.

## Routine

| Day       | Workout     |
|-----------|-------------|
| Monday    | Leg (Quads) |
| Wednesday | Pull        |
| Thursday  | Push        |
| Friday    | Leg (Fem)   |
| Saturday  | Upper       |

## Features

- Browse your 5 workout days from the home screen
- View all exercises per day with set count
- Log weight, reps, and optional notes for each set
- Previous session data is shown as a read-only reference
- Data is saved locally — each new session overwrites the previous one so your numbers stay up to date

## Stack

- Swift / SwiftUI
- SwiftData (local persistence)
- iOS 17+
